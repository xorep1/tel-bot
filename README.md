# 🤖 ربات هوشمند تلگرام (قیمت لحظه‌ای بازار + پاسخ‌دهی خودکار)
 
 یه ربات تلگرام ساده نوشته شده با python که قیمت طلا و دلار را نمایش میدهد با دستور /g و یک سیستم پاسخ خودکار ساده هم براش نوشتم

---

## 🌟 قابلیت‌های اصلی ربات

1. **استخراج لحظه‌ای قیمت‌ها (Scraping):** دریافت کاملاً خودکار و بدون فیلترِ قیمت دلار، مسکوکات (سکه امامی، نیم‌سکه و...)، طلا، ارزهای دیجیتال و نفت از شبکه اطلاع‌رسانی طلا و ارز.
2. **پاسخ به پیام متنی :** واکشن به ارسال کلمه‌ی دقیق.
3. **تشخیص هوشمند خنده:** اسکن محتوای پیام‌ها و تشخیص انواع ایموجی‌های خنده (`😂`, `🤣`, `😅` و...) چه به صورت تکی و چه در میان متن.
4. **پاسخ‌های تصادفی (Random Switching):** انتخاب یک پاسخ چرخشی و تصادفی از میان چندین سناریو برای تنوع در چت.
5. **ریپلای اجباری (Quote Reply):** تمامی پاسخ‌های ربات مستقیماً روی پیام کاربر ریپلای می‌شوند تا ساختار چت در گروه‌ها حفظ شود.
6. **تنظیم خودکار زمان تهران:** نمایش دقیق زمان به‌روزرسانی قیمت‌ها بر اساس افق تهران ($GMT+3:30$) بدون وابستگی به ساعت سرور.
